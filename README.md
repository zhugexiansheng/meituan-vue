# meituan-vue

> 高仿美团的vue项目

## Build Setup

``` bash
# 安装依赖
npm install

# 本地环境运行 serve with hot reload at localhost:8080
npm run dev

# 打包发布 build for production with minification
npm run build

# 打包并且查看分析报告 build for production and view the bundle analyzer report
npm run build --report

# 单元测试 run unit tests
npm run unit

# 端对端测试 run e2e tests
npm run e2e

# 运行所有测试 run all tests
npm test
```
